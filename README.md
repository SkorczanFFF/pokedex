# Pokédex

A React 19 app built with Vite and Tailwind 4 that fetches data from the Pokémon API.

## Installation

```bash
git clone https://github.com/SkorczanFFF/pokedex
npm install
npm run dev
```

## Build

```bash
npm run build
```
